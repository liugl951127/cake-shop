## 五、风险与建议

### 已知风险

| 风险 | 影响 | 缓解 |
|------|------|------|
| 高并发下单可能超卖 | 库存不一致 | 后续用 Redis 分布式锁 + 乐观锁 |
| 大文件上传超时 | 用户体验差 | 后续接 OSS 分片上传 |
| 真实环境与测试环境差异 | bug 漏测 | 灰度发布 + 监控告警 |
| 第三方 API 限流 | 业务受阻 | 加重试 + 降级 + 告警 |

### 业务建议

1. **首推优惠券 + 秒杀组合**, 转化率高
2. **会员日 + 双倍积分**, 提升复购
3. **企业团建蛋糕定制**, B 端新增长点
4. **小程序 + 公众号 + 视频号 三端联动**

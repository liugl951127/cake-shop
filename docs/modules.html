🛒 关键流程 1: 用户下单

1.浏览商品 → 2.加入购物车 → 3.选择地址 → 4.选优惠券 / 积分 → 5.提交订单 → 6.支付成功 → 7.商家接单 → 8.配送 → 9.确认收货

💡 涉及接口: GET /api/goods → POST /api/api/v1/cart → POST /api/api/v1/address → POST /api/api/v1/coupon/mine → POST /api/orders → POST /api/api/v1/pay/wxpay

🎁 关键流程 2: 营销组合

优惠券 + 秒杀价 + 会员折扣 + 积分抵现 + 满减活动 = 最优价

💡 优先级: 秒杀价 > 会员折扣 > 优惠券 > 满减 > 积分抵现

🔒 关键流程 3: 3 层鉴权

请求到达 → InternalRpcFilter → RefererFilter → MiniProgramAuthFilter → JwtAuthFilter → Controller

💡 路径前缀: /api/internal/** → /servicewechat.com → X-Openid+Token → JWT

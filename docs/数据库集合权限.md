# 数据库集合与权限配置(v2.0)

## 集合清单(13 个)

| 集合名 | 说明 | 权限 |
|------|------|------|
| users | 用户信息 | 仅创建者读写 |
| categories | 商品分类 | 所有用户可读,仅管理员可写 |
| goods | 商品信息 | 所有用户可读,仅管理员可写 |
| orders | 订单 | 仅创建者读写,管理员可读 |
| **orderLogs** | 订单流转日志 | 仅云函数可写,创建者/管理员可读 |
| **refunds** | 退款记录 | 仅云函数可写,创建者可读 |
| addresses | 收货地址 | 仅创建者读写 |
| favorites | 收藏 | 仅创建者读写 |
| banners | 轮播图 | 所有用户可读,仅管理员可写 |
| notices | 公告 | 所有用户可读,仅管理员可写 |
| **regions** | 省市区数据 | 所有用户可读,仅管理员可写 |

## 一键配置(云开发控制台 -> 数据库 -> 权限设置)

### users
```json
{ "read": "doc._openid == auth.openid", "write": "doc._openid == auth.openid" }
```

### categories / goods / banners / regions
```json
{
  "read": true,
  "write": "getUserInfo('isAdmin') == true"
}
```

### orders
```json
{
  "read": "doc._openid == auth.openid || getUserInfo('isAdmin') == true",
  "write": "doc._openid == auth.openid || getUserInfo('isAdmin') == true"
}
```

### addresses / favorites
```json
{ "read": "doc._openid == auth.openid", "write": "doc._openid == auth.openid" }
```

### orderLogs / refunds
- 读:仅创建者 / 管理员(自定义规则)
- 写:仅云函数(后端通过 wx.cloud.callFunction 写入)

## 索引建议

云开发控制台 -> 数据库 -> 集合 -> 索引管理:

- **goods**: `{ category: 1, status: 1 }`, `{ recommend: 1, status: 1 }`, `{ name: 1 }`
- **orders**: `{ _openid: 1, status: 1, createTime: -1 }`, `{ status: 1, expireTime: 1 }` (定时器用)
- **orderLogs**: `{ orderId: 1, createTime: 1 }`
- **favorites**: `{ _openid: 1, goodsId: 1 }` (唯一)
- **addresses**: `{ _openid: 1, isDefault: -1 }`
- **regions**: `{ parent: 1, sort: 1 }`
- **refunds**: `{ orderId: 1 }`, `{ outRefundNo: 1 }` (唯一)
